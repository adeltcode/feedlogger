# feedlogger

WordPress plugin to count the number of RSS feed subscribers forked from [EC Studio Inc.](http://techblog.ecstudio.jp/tech-tips/wordpress/feedlogger.html)

## Supported Feed Reader
* Google Reader 
* livedoor Reader 
* Fastladder 
* Yahoo RSS Reader 
* Hatena RSS 
* PAIPO READER 
* Bloglines 
* feedpath Rabbit 
* Netvibes 
* FEEDBRINGER 
* LiveJournal 
* opera 
* Rojo 
* FreshReader 
* DELCO READER 
* NewsGator Online 
* Pluck 
* NewsAlloy 
* RSSFWD 
* Feedshow 
* attensa 
* ThePort 
* POCO 
* Gougou 
* opera 
* Feedeye
